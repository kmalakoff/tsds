## tsds

CLI wrapper for [ts-dev-stack](https://github.com/kmalakoff/ts-dev-stack). Enables running ts-dev-stack commands via npx without local installation.

### Usage

```bash
# Run without installing locally
npx tsds validate
npx tsds build
npx tsds test:node

# Or install globally
npm install -g tsds
tsds validate
```

See [ts-dev-stack](https://github.com/kmalakoff/ts-dev-stack) for full documentation.
